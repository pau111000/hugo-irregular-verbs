Resum Teòric

📌 Què és una funció?

Una funció és una correspondència (o aplicació) que assigna a cada valor de la variable independent x un únic valor de la variable dependent y.

f: x → y = f(x)

Es pot representar mitjançant: una fórmula, una taula de valors o una gràfica.

🇪🇸 Una función asigna a cada valor de x un único valor de y.

📋 Característiques d'una funció

Quan analitzem una funció, estudiem:

  • Domini → interval (on existeix la funció)
  • Recorregut → interval (valors que pren y)
  • Punts de tall → punts (on talla els eixos)
  • Continuïtat / Discontinuïtat → interval
  • Creixement / Decreixement → interval
  • Màxims / Mínims → punts
  • Curvatura → interval
  • Punt d'inflexió (PI) → punt
🇪🇸 Estas son todas las propiedades que analizamos al estudiar una función.

🎯 Domini de definició

El domini d'una funció és el conjunt de punts on la funció existeix i té sentit.

🇪🇸 El dominio es el conjunto de valores de x donde la función existe y tiene sentido.

📐 Polinomis

El domini és tot ℝ (tots els nombres reals).

Exemple: f(x) = x² → Dom f(x) = ℝ = (-∞, +∞)
🇪🇸 Los polinomios siempre tienen dominio ℝ.

√ Radicals

El que hi ha dins l'arrel ha de ser ≥ 0.

f(x) = √g(x) → cal que g(x) ≥ 0
Exemple: f(x) = √x → Dom f(x) = [0, +∞)
Exemple: f(x) = √(x−2) → x−2 ≥ 0 → x ≥ 2 → Dom f(x) = [2, +∞)
🇪🇸 Lo que está dentro de la raíz debe ser ≥ 0.

➗ Racionals (fraccions)

El denominador no pot ser zero. Cal trobar els valors que anul·len el denominador i excloure'ls.

f(x) = P(x)/Q(x) → cal que Q(x) ≠ 0
Exemple: f(x) = 1/(x+3) → x+3=0 → x=−3
Dom f(x) = ℝ − {−3} = (−∞,−3) ∪ (−3,+∞)
Exemple: f(x) = (2x+6)/(x²−3x+2) → x²−3x+2=0 → x=1, x=2
Dom f(x) = (−∞,1) ∪ (1,2) ∪ (2,+∞)
🇪🇸 El denominador no puede ser cero. Encuentra los valores que lo anulan y exclúyelos.

〰️ Continuïtat

Una funció és contínua si la pots dibuixar sense aixecar el llapis del paper.

Si has d'aixecar el llapis, la funció té una discontinuïtat en aquell punt.

🇪🇸 Una función es continua si puedes dibujarla sin levantar el lápiz del papel.

⚡ Tipus de discontinuïtats

Hi ha tres tipus principals:

📊

Salt finit

La funció "salta" d'un valor a un altre. Els dos límits laterals existeixen però són diferents.

🇪🇸 La función "salta" de un valor a otro.

🚀

Salt infinit (asimptòtic)

La funció se'n va a infinit (+∞ o −∞). Hi ha una asímptota vertical.

🇪🇸 La función se va al infinito. Hay una asíntota vertical.

🕳️

Evitable

Hi ha un "forat" a la gràfica. El límit existeix però la funció no està definida o val diferent.

🇪🇸 Hay un "agujero" en la gráfica que se podría rellenar.

🔍 Com identificar-les a la gràfica

Salt finit: Veus un salt entre dos trossos de corba, amb un cercle obert i un tancat a altures diferents.
Salt infinit: La corba puja o baixa verticalment sense límit (es dispara). Línea puntejada vertical = asímptota.
Evitable: Veus un cercle obert (forat) però la corba continua suaument. Es podria "tapar" el forat.

📈 Monotonia (Creixement i Decreixement)

La monotonia descriu si la funció puja o baixa.

f creix quan: si x₁ < x₂ → f(x₁) < f(x₂)

Si avancem cap a la dreta (x augmenta), la funció puja.

f decreix quan: si x₁ < x₂ → f(x₁) > f(x₂)

Si avancem cap a la dreta (x augmenta), la funció baixa.

🇪🇸 Crece: si vamos a la derecha, la función sube. Decrece: si vamos a la derecha, la función baja.

⛰️ Màxims i Mínims

Un màxim és un punt on la funció passa de créixer a decréixer (punt més alt local).

Un mínim és un punt on la funció passa de decréixer a créixer (punt més baix local).

Màxim: 📈 → ⛰️ → 📉 (puja i després baixa)
Mínim: 📉 → 🏜️ → 📈 (baixa i després puja)
🇪🇸 Máximo: sube y luego baja. Mínimo: baja y luego sube.

📍 Recorregut (Rang)

El recorregut és el conjunt de valors que pren y = f(x). Mirem la gràfica de baix a dalt.

Exemple: Si la gràfica va des de y = −3 fins a y = +∞ passant per un forat, pot ser: Rec f(x) = (−∞, −3) ∪ [4, +∞)
🇪🇸 El recorrido (rango) son todos los valores que toma y.

✂️ Punts de tall amb els eixos

Tall amb eix X: Fem y = 0 i resolem. Els punts tenen forma (a, 0).

Tall amb eix Y: Fem x = 0 i calculem f(0). El punt té forma (0, b).

Exemple: Tall eix X: (−10, 0) i (7, 0) — Tall eix Y: (0, 3)
🇪🇸 Corte con eje X: y=0. Corte con eje Y: x=0.

📏 Notació d'intervals

Recordem com escriure intervals:

(a, b) obert
[a, b] tancat
[a, b) semiobert
(a, b] semiobert

( ) = el punt no s'inclou (obert, cercle buit)

[ ] = el punt sí s'inclou (tancat, cercle ple)

∞ i −∞ sempre van amb parèntesi ( ) perquè no es poden assolir.

🇪🇸 ( ) = no incluido (abierto). [ ] = incluido (cerrado). ∞ siempre con paréntesis.
